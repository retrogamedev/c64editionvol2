CharPad 2.3 User Manual - Subchrist Software, 2019.

Program Features.

- Character image editing in multi-colour and high-resolution modes.

- Character material editing for creating collision detection data.

- Tile editing (rectangular arrangements of character images) from 1x2/2x1 up to 8x8.

- Pixel-level tile editing utilising unique compression/decompression techniques.

- Map editing using character or tile sets.

- Unique 'Material Vision' rendering mode for immediate viewing of collision data.

- I/O support for the Shoot 'em Up Construction Kit (SEUCK).

- Support for ripping data from emulator (VICE x64) snapshots.

- Accurate bitmap image import/analysis/conversions.

- Reference auto-correction for delete/insert/cut/paste operations on sets.

- Multiple levels of Undo/Redo.
