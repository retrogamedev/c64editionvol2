CharPad 2.3 User Manual - Subchrist Software, 2019.

The Character Image Editor Form.

The Character Image Editor form allows you to edit the currently selected character image.

Painting

With one of the pen tools selected, pixels can be drawn using the mouse.

The left button draws using the currently selected colour (on the Project form) and the right button draws using the transparent colour.

With the flood-fill tool selected, areas of the image can be flood-filled, clicking the left mouse button while over the drawing canvas will flood-fill using the currently selected colour and the right button will flood-fill using the transparent colour.

Transformations

The following transformations are available when this form is active...

-Scroll (up, down, left, right).

-Flip (left-to-right, top-to-bottom).

-Reflect (left-to-right, top-to-bottom).

-Negative.

-Colour swapping (for multi-colour images).
