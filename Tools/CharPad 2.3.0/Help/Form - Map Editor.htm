CharPad 2.3 User Manual - Subchrist Software, 2019.

The Map Editor Form.

The Map Editor form allows you to view and edit the project's map.

Resizing the map

The form's toolbar contains numeric entry boxes for displaying and altering the map's width and height, to adjust the map's dimensions simply type new values into these boxes and hit enter.

nb. The smallest possible map size is 1x1, the largest is 8192 x 8192.

Painting

With one of the brush tools selected, map cells can be painted using the mouse.

The left button paints using the currently selected character image (or tile) and the right button draws using character image (or tile) zero.

With the flood-fill tool selected, areas of the map can be flood-filled, clicking the left mouse button while over the drawing canvas will flood-fill using the currently selected character image (or tile) and the right button will flood-fill using character image (or tile) zero.

Selecting map areas

With the Selection tool enabled, clicking on the map the right mouse button will select a map cell.

By clicking on a second map cell with the SHIFT key held down, a rectangular area of the map can be selected (it will appear highlighted).

Transformations

The following transformations are available when this form is active...

-Scroll (up, down, left, right).

-Flip (left-to-right, top-to-bottom).

-Reflect (left-to-right, top-to-bottom).

Notes:-

-When the Map form is active, the main form offers the Scroll, Flip and Reflect operations, these can be performed on the whole map, or if the the Selector tool is enabled, the operations can be performed on just a currently selected area.
