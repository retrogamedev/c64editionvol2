CharPad 2.3 User Manual - Subchrist Software, 2019.

The Character Set Form.

The Character Set form allows you to view and organise character images and also perform multi-image transformations.

Resizing the character set

To set the quantity of images in the character set, simply type a new quantity into the numeric entry box on the form's toolbar and hit enter.

nb. The minimum quantity is 1 and the maximum is 65536.

Selecting images

To select a single image, just click it using the left mouse button, it will appear highlighted.

To select a range of images, first select a primary image as the first in a desired range, then select a secondary image whilst holding down the SHIFT key.

The selected range will appear highlighted.

Transformations

The following transformations are available when this form is active...

-Scroll (up, down, left, right).

-Flip (left-to-right, top-to-bottom).

-Reflect (left-to-right, top-to-bottom).

-Negative.

-Colour swapping (for multi-colour images).

Other functions

-Swapping - By selecting one image using the left mouse button and then selecting a second image using the right mouse button, the image's positions in the set can be swapped.

-Cropping - With a range of images selected, the 'Crop' menu item will remove all images not included in the selection.

-Cycling - With a range of images selected, the 'Cycle Backwards' and 'Cycle Forwards' menu items will cycle (rotate) the selected image's positions in the set by one place.

nb. Image cycling does not perform correction on ie. map data, so the function can be used to produce an animation effect.
